🔧 기능 설명

▶ 게임 기본 구조

터미널(콘솔) 환경에서 실행되는 ASCII 기반 레이싱/회피 게임

트랙이 위에서 아래로 스크롤되며 플레이어는 좌우 이동만 가능

목표 지점(GOAL)에 도달하면 게임 클리어, 충돌 시 게임 오버


▶ 게임 모드 구성

Normal Mode

기본 난이도의 게임 모드

장애물 충돌 시 게임 오버

아이템 획득을 통한 점수 시스템 적용

트랙 기반 진행 구조

Hard Mode

Normal Mode를 기반으로 난이도 강화

미사일 시스템 추가

미사일에 피격 시 즉시 게임 오버

더 빠른 진행 속도와 높은 난이도



▶ 조작 시스템

키보드 입력 기반 조작

좌우 이동: A / D 또는 ← / →

게임 종료: ESC

모드 선택 화면에서 방향키 + 스페이스로 선택



▶ 트랙(맵) 시스템

ASCII 문자로 구성된 트랙 맵 사용

트랙은 파일(track.txt)로 분리되어 관리

트랙이 위로 스크롤되며 진행 상황이 반영됨

벽(│)을 기준으로 주행 가능 영역 자동 판별



▶ 플레이어 시스템

플레이어는 ASCII 아트로 표현된 로켓/차량 형태

좌우 이동만 가능하며 상하 위치는 고정

플레이어와 장애물/미사일/아이템의 좌표 충돌을 통해 이벤트 처리



▶ 장애물 시스템 (Normal 중심)

트랙 내부에 랜덤 위치로 장애물 생성

장애물은 트랙 위에 직접 삽입(스탬프 방식)

플레이어가 장애물과 충돌 시 즉시 게임 오버



▶ 미사일 시스템 (Hard 전용)

Hard Mode에서만 활성화되는 공격 요소

일정 주기로 미사일 생성

미사일은 화면 우측에서 좌측으로 이동

플레이어와 충돌 시 즉시 게임 오버



▶ 아이템 시스템

게임 진행 중 랜덤 아이템 생성

아이템 종류 및 효과

★ : 점수 +5

● : 점수 +3

▲ : 점수 -2

플레이어와 아이템 충돌 시 즉시 점수 반영

아이템 획득 후 화면에서 제거



▶ 점수 시스템

시간 경과에 따라 점수 증가

아이템 획득 시 추가 점수 반영

Normal / Hard 모드 각각 점수 시스템 적용

최고 점수(하이스코어) 파일로 저장



▶ 하이스코어 저장 기능

모드별 하이스코어 파일 관리

Normal: highscore_normal.txt

Hard: highscore_points.txt

게임 종료 시 현재 점수가 최고 점수보다 높을 경우 자동 갱신

다음 실행 시 이전 최고 점수 로드



▶ 미니맵 시스템

화면 우측에 미니맵 표시

전체 트랙 대비 현재 진행 위치를 시각적으로 표현

화면 크기에 맞게 자동 스케일 조정

플레이어 진행 상황을 한눈에 확인 가능



▶ 게임 시작 연출

게임 시작 전 3 → 2 → 1 → START! 카운트다운 출력

ASCII 아트를 활용한 시각적 연출

카운트다운 종료 후 게임 자동 시작


▶ UI 출력 및 화면 처리

터미널 크기에 맞춘 중앙 정렬 UI

깜빡임을 줄이기 위한 커서 제어 방식 사용

상태 UI(점수, 속도, 진행도 등)를 실시간 표시

사이드바 형태의 정보 패널 구성



▶ 안정적인 종료 처리

게임 종료 시 키보드 후킹 해제

비정상 종료 없이 안전하게 프로그램 종료

ESC 입력 또는 게임 오버 시 정상 종료 루트 제공
