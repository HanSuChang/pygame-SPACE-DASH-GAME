# 🚀 SPACE DASH

> 터미널에서 즐기는 ASCII 기반 레이싱/생존 게임

---

## 📌 게임 소개

**SPACE DASH**는 콘솔(터미널) 환경에서 실행되는 ASCII 아트 기반 레이싱 게임입니다.  
트랙이 위로 스크롤되는 동안 플레이어는 좌우로 이동하며 장애물을 피하고, GOAL 라인 도달 시 클리어됩니다.

---

## 🎮 게임 모드

### NORMAL 모드
- 기본 난이도
- 장애물(바위/콘) 충돌 시 GAME OVER
- 아이템 점수 시스템 적용 (★ +5 / ● +3 / ▲ -2)
- 최고점수 저장 (`highscore_normal.txt`)

### HARD 모드
- NORMAL 구조 기반 + 난이도 강화
- **미사일 시스템 추가** — 피격 시 즉시 GAME OVER
- 최고점수 저장 (`highscore_points.txt`)

---

## 🕹️ 조작법

| 키 | 동작 |
|---|---|
| `A` / `←` | 좌측 이동 |
| `D` / `→` | 우측 이동 |
| `ESC` | 게임 종료 |

---

## ✨ 주요 기능

| 기능 | 설명 |
|---|---|
| ASCII 트랙 | 파일 기반 트랙 데이터 (`track.txt`, `track_hard.txt`) |
| 카운트다운 연출 | 3 → 2 → 1 → START! 시작 연출 |
| 인게임 미니맵 | 우측 사이드바에 전체 트랙 진행도 표시 |
| 아이템 시스템 | 도로 영역에 랜덤 스폰, 점수 가감 |
| 미사일 시스템 | HARD 전용, 셀 집합 교집합 방식 충돌 판정 |
| 점수 시스템 | 시간 누적 + 아이템 점수 |
| 하이스코어 | 모드별 최고점수 파일 저장 |

---

## 🛠️ 사용 기술

- **언어**: Python 3
- **핵심 모듈**: `time`, `shutil`, `unicodedata`, `keyboard`
- **화면 처리**: ANSI 제어코드, 커서 숨김/이동
- **충돌 판정**: `set` 자료형 기반 좌표 셀 교집합
- **맵 구조**: 파일 기반 ASCII 트랙, 2D 리스트 grid 렌더링

### 🔍 A (한수창) 상세 기여

**HARD 모드 미사일 시스템** (핵심 구현)
- 문자열 스프라이트(`<====`) 기반 미사일 표현
- 미사일 생성 → 이동 → 화면 밖 자동 제거 흐름 구현
- 절대좌표(`abs_y`) 관리로 스크롤과 자연스럽게 연동
- 플레이어/미사일 좌표를 각각 `set`으로 변환 후 교집합으로 충돌 판정

**ASCII 트랙맵 제작**
- Normal / Hard 모드용 트랙 데이터 직접 제작
- 벽(`│`)과 도로 영역을 고려한 레이아웃 설계
- 장애물·아이템·미사일 배치 가능한 구조로 구성

**인게임 UI (50% 기여)**
- 3 → 2 → 1 → START! 카운트다운 연출 제작
- ASCII 아트 큰 글자를 트랙 화면에 합성하여 출력
- 인게임 미니맵 UX 아이디어 제안 및 구조 반영
- 시간 + 아이템 기반 점수 시스템 확장 설계
